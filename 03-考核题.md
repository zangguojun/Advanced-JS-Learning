#### 4

> 因为`()`包裹了`{} + 0`，所以认为`{}`和`0`是一个整体，结果就为`[object Object]0`

```js
console.log(0 + {}, '0[object Object]');
// 交互模式
// {}+0
// 0
// ({})+0
// "[object Object]0"
// ({}+0)
// "[object Object]0"

{} + 0
{var a = 1} + 0
console.log(`{} + 0,{var a = 1} + 0以及function a(){var a = 1} + 0在交互模式下为0`);
```
